Track Finder Bookmarklet

Highlight a tracklist on any page, click the bookmarklet, land on search links for every track. Works on 1001tracklists, MixesDB, YouTube descriptions, Reddit threads, anywhere really.

Install

  1. Show your bookmarks bar (Chrome/Edge: Ctrl+Shift+B, Firefox: Ctrl+B, Safari: ⌘+Shift+B)
  2. Drag the orange button below to your bookmarks bar
Find Tracks
Drag the button up to your bookmarks bar. Don't click it here.

Use it

  1. Open a tracklist page (1001tracklists, MixesDB, a SoundCloud set description, a Reddit thread, wherever)
  2. Highlight the tracks with your cursor (skip the page headers, comments, ads)
  3. Click the Find Tracks bookmark
  4. Track Finder opens in a new tab with your tracks parsed and the first platform's search links ready
Tip: The better your selection, the cleaner the output. If you don't select anything, the bookmarklet grabs the whole page text — which works but pulls in navigation and ads that the parser has to filter. Selection-first is faster.

1001tracklists specifically

Cloudflare Turnstile blocks server-side scraping, so this bookmarklet is the cleanest path. Since it runs inside your already-verified browser tab, it sees the tracks the site actually rendered. Two recommended flows:

  • Select the tracklist section (drag from the first track to the last), then click Find Tracks
  • Or use 1001tracklists' Export Tracklist button, select the exported text, click Find Tracks

Mobile

iOS Safari and Android Chrome both support bookmarklets, just with extra steps:

  • iOS Safari: bookmark any page, edit the bookmark, replace the URL with the bookmarklet code (shown below), save. Then tap it from any page.
  • Android Chrome: same pattern — bookmark a page, edit, paste the code as the URL.
View the bookmarklet source 
javascript:(function(){
  var s=window.getSelection().toString().trim();
  var t=s||((document.body.innerText||document.body.textContent||'').trim());
  if(!t){alert('Track Finder: nothing to scrape.');return;}
  var n=document.title.trim();
  var u='https://track-finder.vercel.app/?'+new URLSearchParams({tracks:t,name:n,go:'1'}).toString();
  if(u.length>7500){alert('Track Finder: selection too large ('+t.length+' chars). Highlight a smaller section.');return;}
  window.open(u,'_blank');
})();
